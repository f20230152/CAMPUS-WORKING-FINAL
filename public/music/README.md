The file must be named exactly: **`background-music.mp3`**

## Adding Your Music File

1. **Find a lively, upbeat track** that matches the Swiggy Wrapped vibe:
   - Upbeat, energetic
   - Looping-friendly (seamless loop)
   - 2-3 minutes long (will loop automatically)
   - Royalty-free or licensed for use

2. **Recommended sources:**
   - [Pixabay Music](https://pixabay.com/music/) - Free, no attribution needed
   - [Free Music Archive](https://freemusicarchive.org/)
   - [YouTube Audio Library](https://www.youtube.com/audiolibrary)
   - [Incompetech](https://incompetech.com/music/) - Kevin MacLeod's music

3. **File formats:**
   - Primary: `background-music.mp3` (MP3 format) - **REQUIRED**
   - Fallback: `background-music.ogg` (OGG format, optional)

4. **File location:**
   - **Required:** `public/music/background-music.mp3`
   - **Optional:** `public/music/background-music.ogg` (fallback if MP3 fails)

## Recommended Music Style

- **Genre:** Upbeat electronic, pop, or energetic instrumental
- **Tempo:** 120-140 BPM (moderate to fast)
- **Mood:** Celebratory, fun, energetic
- **Duration:** 2-3 minutes (will loop seamlessly)
